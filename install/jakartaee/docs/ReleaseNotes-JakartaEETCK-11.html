Known Problems and Workarounds

Running the TCK Against Jakarta EE Platform 11 Compatible Implementation on Windows

When configuring and running the Jakarta EE Platform TCK against the Jakarta EE Platform 11 compatible implementation on Windows, please make sure to install all related software (Java SE, Jakarta EE 11 compatible implementation, the Platform TCK, etc.) on the same drive (for example, C:). If you want to install on a drive other than C:, please be sure to change the following properties in <TS_HOME>/bin/ts.jte:

pathsep=;
ri.applicationRoot=c:
s1as.applicationRoot=c:
