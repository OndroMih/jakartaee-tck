Jakarta EE Messaging Technology Compatibility Kit, Version 3.1

Release Notes, December 2021

Kit Contents

The Jakarta EE Messaging, Version 3.1 Technology Compatibility Kit (TCK) includes the following items:

  • Jakarta EE Messaging TCK tests signature, API, and End-to-End tests:

    • A signature test that checks that all of the public APIs are supported in the Jakarta EE Messaging Version 3.1 implementation that is being tested

    • API tests for all of the public APIs under the jakarta.jms package

    • End-To-End tests that demonstrate sending/receiving Jakarta EE Messaging messages to/from a QUEUE or TOPIC

Jakarta EE Messaging TCK Facts

The test suite bundle contains the following:

  • 904 total tests

Platform Notes

The Jakarta EE Messaging TCK tests have been built with JDK 11 and tested with JavaTM Platform, Standard Edition 11 and 17.

The Jakarta EE Messaging TCK tests have been run on the following platforms:

  • Debian Linux 10

The Jakarta EE Messaging TCK tests have been run against a Jakarta Messaging 3.1 compatible implementation.

Installing, Setting Up, and Running the Jakarta EE Messaging TCK

Refer to the Jakarta EE Messaging TCK 3.1 User's Guide for complete instructions on installing, setting up, and running the Jakarta EE Messaging TCK.
