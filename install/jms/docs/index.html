Guide to Jakarta EE Messaging TCK 3.1 Documentation

The Jakarta EE Messaging TCK 3.1 documentation includes the following:

  • The Jakarta EE Messaging TCK Release Notes provides the information on the test harness, the test suite and the platforms tested.
  • Release Notes are included.
    • If updates are necessary, revised Release Notes may be provided through the Jakarta EE Messaging specification web site -- https://jakarta.ee/specifications/messaging/3.1/.
  • The Jakarta EE Messaging Technology Compatibility Kit, Version 3.1 User's Guide (HTML, PDF) provides the information that you need to install, set up, and run the Jakarta EE Messaging TCK, Version 3.1. In addition, the guide provides the rules you must comply with to pass the Jakarta EE Messaging TCK.
  • The Javadoc Assertion List lists all the javadoc assertions that are tested by the Jakarta EE Messaging TCK.
  • The Specification Assertion List lists all the specification assertions that are tested by the Jakarta EE Messaging TCK.
